# Security Policy

## Scope

This repository contains prompt-based configuration files (markdown, JSON, YAML, Python validation scripts). It does not contain application code that runs in production environments.

## Reporting

If you discover a security concern (e.g., leaked credentials, prompt injection vectors in skill/agent definitions), please report it by opening a GitHub issue or contacting the maintainer.

For issues related to Claude Code itself, report to [Anthropic](https://github.com/anthropics/claude-code/issues).
